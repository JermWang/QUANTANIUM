Welcome to Quantanium’s official GitHub repository! Quantanium is at the forefront of leveraging quantum computing to enhance the security of Web3 wallets and decentralized applications. This repository contains an overview of our technologies, mechanics, and innovations to foster trust and resilience in the decentralized world.
